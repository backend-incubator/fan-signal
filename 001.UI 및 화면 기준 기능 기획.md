## 화면 기준 기능 기획

1. 회원
   1. 회원 가입
   2. ~~회원 탈퇴~~
   3. 로그인 / 로그아웃
2. Home
   1. 게시글 조회
   2. 게시글 검색
   3. 게시글 좋아요 등록/취소
   4. 게시글 북마크 등록/취소
3. Common
   1. 댓글 작성/삭제
4. Ticket
   1. 티켓 조회
   2. 티켓 구매
5. Post
   1. 게시글 등록 : 1장 이상의 사진과 텍스트 필수
6. User
   1. My
      1. 프로필 변경
      2. 티켓 구매 모아보기
      3. Bookmark 모아보기
   2. Other
      1. 팔로우/메시지 버튼
   3. Common
      1. User 정보 조회
      2. 팔로워/팔로잉 목록
      3. 게시글 모아보기
7. DM
   1. DM방 목록 조회
   2. User 검색 : 내 팔로잉 기준
   3. DM방 생성
   4. DM방 삭제
   5. DM 전송
   6. DM 조회
   7. ~~DM 검색~~
8. 알림
   1. 내 게시글 좋아요 알림
   2. 내 게시글 댓글 등록 알림
   3. 채팅 알림
9. 게시글 설정
   1. 게시글 삭제
   2. 게시글 수정
