# 005. 게시글 API 명세서

## 게시글 등록

- Method : POST
- URL : /v1/api/post
- RequestBody 

  | Field     | Type   | Description | 
  |:----------|:-------|:------------|
  | member_id | String | 사용자 고유 ID   |
  | content   | String | 게시글 내용      |
  | photos    | Array  | 사진 목록       |
- RequestBody Example
  ```json
  {
    "member_id": "memberUniqueId",
    "content" : "TEST 글을 작성해봅니다.",
    "photos": ["https://avatars.githubusercontent.com/u/46712693?v=4"]
  }
  ```
- ResponseBody

  | Field   | Type   | Description |
  |:--------|:-------|:------------|
  | status  | Int    | HTTP 상태 코드  |
  | message | String | 상태 메시지      |
- ResponseBody Example
  ```json
  {
    "status": 204,
    "message": "success"
  }
  ```

## 게시글 조회

- Method : GET
- URL : /v1/api/post/{id}
- Request Parameter

  | Field     | Type   | Description | 
  |:----------|:-------|:------------|
  | id        | String | 게시글 고유 ID   |
  | member_id | String | 사용자 고유 ID   |

- ResponseBody

  | Field   | Type   | Description |
  |:--------|:-------|:------------|
  | status  | Int    | HTTP 상태 코드  |
  | message | String | 상태 메시지      |
  | data    | Object | 응답 데이터      |

  - data

    | Field     | Type    | Description    |
    |:----------|:--------|:---------------|
    | id        | String  | 게시글 고유 ID      |
    | content   | String  | 게시글 내용         |
    | photos    | Array   | 사진 목록          |
    | author    | String  | 게시글 작성자 ID     |
    | author_id | String  | 게시글 작성자 고유 ID  |
    | likeCount | Int     | 게시글 좋아요 수      |
    | chatCount | Int     | 게시글 채팅 수       |
    | isFollow  | Boolean | 게시글 작성자 팔로잉 여부 |

- ResponseBody Example
  ```json
  {
    "status": 204,
    "message": "success",
    "data": {
      "id": "postUniqueId",
      "content" : "TEST 글을 작성해봅니다.",
      "photos": ["https://avatars.githubusercontent.com/u/46712693?v=4"],
      "author": "member",
      "author_id": "memberUniqueId",
      "likeCount": 3,
      "chatCount": 0,
      "isFollow": true
    }
  }
  ```


## 게시글 목록 조회 및 검색

- Method : GET
- URL : /v1/api/post
- Request Parameter

  | Field     | Type   | Description | 
  |:----------|:-------|:------------|
  | from      | Int    | 조회 시작 위치    |

- ResponseBody

  | message | String | 상태 메시지      |
  | data    | Array  | 응답 데이터      |

  - data

    | Field     | Type    | Description    |
    |:----------|:--------|:---------------|

    | photos    | Array   | 사진 목록          |
    | author    | String  | 게시글 작성자 ID     |
    | author_id | String  | 게시글 작성자 고유 ID  |
    | likeCount | Int     | 게시글 좋아요 수      |
    | chatCount | Int     | 게시글 채팅 수       |
    | isFollow  | Boolean | 게시글 작성자 팔로잉 여부 |

- ResponseBody Example

  {
    "status": 204,
